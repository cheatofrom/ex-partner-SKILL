# 基础信息录入脚本

## 开场白

```
我来帮你创建这位前任的 Skill。只需要回答 3 个问题，每个都可以跳过。
```

---

## 问题序列

### Q1：前任称呼

```
这位前任怎么称呼？（昵称、代号或名字都行，多个字用 - 连接）

例：qing-yun
```

- 接受任意字符串
- 生成的 slug 统一用 `-` 连接（不用下划线）
- 中文自动转拼音再用 `-` 连接（"青云" → `qing-yun`，"小李" → `xiao-li`）
- 英文直接小写 `-` 连接（"Big Mike" → `big-mike`）

---

### Q2：恋爱基本信息

把恋爱时长、分手原因、年龄、性别放在一个问题里，让用户一句话说完：

```
用一句话描述这段关系的基本信息——恋爱时长、分手原因、年龄、性别，想到什么写什么，跳过也行。

例：在一起3年 异地恋分手 26岁 男
```

从用户的回答中解析以下字段（缺失的留空）：
- **恋爱时长**
- **分手原因**
- **年龄**
- **性别**

#### 分手原因常见类型参考

| 大类 | 具体原因 | 表现形式 |
|------|---------|------------|
| 现实阻碍 | 异地/距离、家庭反对、经济压力 | 无法陪伴、长辈不看好、彩礼/买房分歧 |
| 感情淡化 | 缺乏激情、冷暴力、沟通不畅 | 没话说、不回消息、互相敷衍 |
| 核心冲突 | 三观不合、性格不合、未来规划不同 | 频繁吵架、消费观差异、发展城市不同 |
| 原则性问题 | 出轨/劈腿、欺骗、黄赌毒 | 发现聊骚、隐瞒恋爱史、底线被触碰 |

### Q3：恋爱性格画像

把 MBTI、星座、恋爱依恋类型、核心矛盾点、主观印象全部合在一起，让用户自由描述：

```
用一句话描述他在恋爱中的性格——MBTI、星座、个性特点、恋爱依恋类型、你对他的印象，
想到什么写什么，跳过也行。

例：INTJ 摩羯座 回避型依恋 冷暴力高手 吵架从来不解释原因
```

从用户的回答中识别并提取以下字段（缺失的留空）：
- **MBTI**：16 种标准类型
- **星座**：12 星座
- **个性标签**：从下方标签库匹配，也接受自定义描述
- **依恋类型**：安全型 / 焦虑型 / 回避型 / 恐惧型
- **主观印象**：无法归类的自由描述，直接保留原文

#### 个性标签库

**恋爱态度**：认真专一 / 骑驴找马 / 渣男渣女 / 海王海后 / 恋爱脑 / 极其理智

**沟通风格**：有话直说 / 让你猜 / 喜欢冷战 / 爱发小作文 / 只读不回 / 夺命连环call

**情绪风格**：情绪稳定 / 玻璃心 / 容易暴怒 / 阴晴不定 / 表面和气 / 阴阳怪气

**话术与手段**：PUA 高手 / 情感操控 / 画大饼专家 / 倒打一耙 / 爱讲大道理 / 情绪勒索
